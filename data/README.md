# Case-Study Data

This directory contains the input and output workbooks associated with the RC-ES MADM III computational case study.

The files are provided to support transparency, reproducibility, and independent inspection of the case-study structure and exported computational results.

## Directory structure

```text
data/
├── input/
│   └── RC_ES_MADM_III_Case_Study_Input.xlsx
└── output/
    └── RC_ES_MADM_III_Case_Study_Output.xlsx
```

## Input workbook

The `input/` directory contains the structured Excel workbook used by the RC-ES MADM III Decision Studio.

The input workbook includes the case-study alternatives, evaluation criteria, criterion directions, scenario definitions, scenario probabilities, scenario-specific subjective weights, raw performance values, and computational settings required to execute the case study.

## Output workbook

The `output/` directory contains the exported results generated by the RC-ES MADM III Decision Studio.

The output workbook includes scenario-level diagnostics, alternative scores and rankings, rank-stability outputs, first-rank probability information, alternative-distinction diagnostics, scenario-weighted portfolio results, and benchmark-related outputs used for comparative interpretation.

## Reproducibility note

The workbooks are provided as reproducibility material for the computational case study. They do not contain the full manuscript, unpublished derivations, or complete mathematical exposition of the RC-ES MADM III framework.
