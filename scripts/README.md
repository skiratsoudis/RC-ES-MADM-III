# Figure-Reproduction Scripts

This directory contains the standalone R script used to reproduce the main computational figures associated with the RC-ES MADM III case study.

## Main script

```text
RC_ES_MADM_III_V.1.0.0_Figures_3_to_6_EMBEDDED_DATA.R
```

## Purpose

The script reproduces the result-based figures generated from the RC-ES MADM III computational case study. It is intended to support transparency, reproducibility, and independent inspection of the visual outputs used to interpret the model results.

The script produces the following figures:

* **Fig. 3**: Scenario-level diagnostic profile;
* **Fig. 4**: Scenario-specific DRI ranking transitions;
* **Fig. 5**: Scenario-weighted DRI portfolio ranking;
* **Fig. 6**: Spearman rank correlations with benchmark MADM methods.

## Standalone design

The script is fully standalone. The plotting data required to reproduce the figures are embedded directly in the R script. Therefore, the script does not require reading the Excel input or output workbooks.

This design ensures that the figures can be reproduced independently from the main Shiny application and without additional data-processing steps.

## How to run

From the root directory of the repository, run:

```r
source("scripts/RC_ES_MADM_III_V.1.0.0_Figures_3_to_6_EMBEDDED_DATA.R")
```

Alternatively, open the script directly in RStudio and run it.

## Output

The script creates a local output directory and exports the generated figures in publication-ready format.

The following export settings can be modified inside the script:

* output directory;
* image format;
* resolution;
* figure width and height;
* base font size;
* label size.

## Notes

Fig. 1 and Fig. 2 are conceptual workflow and case-study architecture diagrams. They are not generated by this script and are prepared separately as schematic figures.

The script is provided as reproducibility material for the computational case study and does not disclose the full unpublished mathematical formulation of the RC-ES MADM III framework.
